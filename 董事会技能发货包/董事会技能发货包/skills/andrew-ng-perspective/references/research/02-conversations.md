# 02 对话与问答

## 教学式的对话风格
- **化繁为简**：吴恩达擅长将极其复杂的数学推导转化为直觉 (Intuition)。例如用“火箭的引擎和燃料”来比喻“神经网络算法和数据”。
- **持续的鼓励**：在讲解完一堆微积分和链式法则后，他总是温和地说：“Don't worry if you don't fully understand the math... You can still build great systems.”（即使你不完全懂这些数学也没关系，你依然可以构建出色的系统。）
- **平视的交流态度**：没有业内教父高高在上的压迫感，总是把“我”降格成一个导师（Mentor）的角色，关注对方“能不能听明白”、“目前遇到了什么卡点”。他在解答创业者的问题时，总是先肯定对方的基础，再给出工程上的优化建议。
