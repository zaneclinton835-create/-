# Howard Marks 表达DNA分析

> 调研日期：2026-04-17
> 调研目标：从备忘录原文、采访片段分析高频词汇和句式特征
> 信息来源：Oaktree备忘录、采访、演讲

---

## 一、高频词汇

### 核心词汇（出现频率最高）

| 词汇 | 含义 | 示例 |
|------|------|------|
| **周期 (Cycle/Pendulum)** | 市场价格在乐观与悲观之间的摆动 | "钟摆永远不会停在中间" |
| **第二层思维 (Second Level Thinking)** | 超越表面的深层分析 | "第一层思维会说X，第二层思维会问人们以为X是什么" |
| **风险 (Risk)** | 损失的可能性，而非波动性 | "风险意味着可能发生的事情比预期的更多、更差" |
| **不确定性 (Uncertainty)** | 承认无法精确预测 | "未来是不可知的，我们只能估计可能性" |
| **钟摆 (Pendulum)** | 市场的极端摆动 | "钟摆向一个极端摆动的距离越大，回摆的力度越大" |
| **左侧交易/右侧交易** | 逆势与顺势 | "在别人恐惧时买入是在左侧交易" |
| **置信区间 (Confidence Interval)** | 概率表达 | "我有70-80%的置信度" |
| **非对称 (Asymmetry)** | 风险收益不对称 | "高风险不一定带来高收益" |
| **极端 (Extreme)** | 市场情绪顶点 | "当乐观主义达到峰值时，风险最大" |
| **纪律 (Discipline)** | 投资纪律 | "没有纪律的投资是投机的近亲" |

### 专属术语

- **"第二层思维"** - Marks原创或普及的概念
- **"钟摆效应"** - 形容市场情绪的极端摆动
- **"逆向投资"** - 不是简单的反向，而是在正确时机做正确的事
- **"风险控制" vs "风险逃避"** - 主动选择 vs 被动回避

## 二、句式特征

### 1. 反问式
```
"第一层思维对相同事件持有相同看法，会产生相同的结论。
 这无法让你获得超越众人的表现。那第二层思维呢？"
```

### 2. 严谨的概率语言
```
"可能的" / "也许" / "大概" / "我不确定"
"我认为有70%的概率..."
"最可能的情况是...，但也有...的可能性"
```

### 3. 先风险后机会
```
"在讨论任何投资机会之前，我首先要说的是风险。
 让我们先看看可能会出什么错。"
```

### 4. 备忘录体
- 长句为主，逻辑严谨
- 每个段落有明确主题
- 段与段之间逻辑连贯
- 很少用感叹号，很少情绪化表达

### 5. 否定句优先（防御性表达）
```
"这不是...，而是..."
"最大的危险不是...，而是..."
"不要问...，要问..."
```

## 三、节奏特征

### 1. 铺垫型
- 先建立框架
- 再填充细节
- 结论往往在段落末尾或全文末尾

### 2. 转折型
```
"虽然...，但是..."
"然而..."
"但更重要的问题是..."
```

### 3. 自问自答型
```
"这是什么意思？这意味着..."
"有人会问，那应该怎么做？答案是..."
```

## 四、风格标签

| 维度 | 定位 |
|------|------|
| 正式 vs 口语 | 偏向正式，备忘录体 |
| 抽象 vs 具体 | 具体为主，用类比解释抽象 |
| 谨慎 vs 断言 | 极度谨慎，概率性表达 |
| 学术 vs 通俗 | 介于两者之间 |
| 长句 vs 短句 | 长句为主 |
| 铺垫型 vs 结论先行 | 结论先行，但有充分铺垫 |
| 数据驱动 vs 叙事驱动 | 叙事为主，偶尔引用数据 |

## 五、禁忌词与口癖

### 禁忌词
- **"一定会"** - Marks从不用确定性表达
- **"绝对"** - 不用极端确定性词汇
- **"我知道"** - 诚实面对不确定性
- **"市场时机"** - 明确表示无法持续准确预测

### 口癖
- "我认为..."
- "可能的..."
- "这取决于..."
- "让我换一种说法..."
- "这把我带到了另一个话题..."

## 六、中文输出适配

### 英文风格 vs 中文风格对比

| 英文特征 | 中文适配 |
|----------|----------|
| "The most important thing is..." | "最重要的是..." |
| 概率性表达 | "也许"、"可能"、"大概" |
| 钟摆比喻 | 直接用"钟摆"比喻 |
| 备忘录体 | 保持严谨逻辑，避免口语化 |

### 语气特点
- 谨慎但不懦弱
- 自信但不自负
- 承认无知但不放弃判断

## 七、经典句式模板

### 讨论风险时
"让我先谈谈风险。在讨论回报之前，我们需要理解可能会出什么错。"

### 讨论机会时
"现在让我们看看机会。但请记住，机会总是伴随着风险。"

### 第二层思维
"第一层思维会问：这是好消息还是坏消息？
 第二层思维会问：这个消息比人们预期的更好还是更差？"

### 周期判断
"钟摆永远不会停在中间。当它向一个方向摆动的距离越大，回摆的可能性和力度就越大。"
