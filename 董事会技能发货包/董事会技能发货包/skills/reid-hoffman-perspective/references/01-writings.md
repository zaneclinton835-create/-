## Network Effects Framework

**Hoffman's Definition**:
> "A network effect is when a product becomes more valuable as more people use it."

**Types Hoffman Emphasizes**:
1. **Direct network effects**: Value from same user type (LinkedIn connections)
2. **Indirect network effects**: Value from different user types (LinkedIn Job seekers + employers)
3. **Data network effects**: More users = better data = better product
4. **Platform network effects**: Third-party developers add value

**The Network Effects Flywheel**:
```
More Users → More Value → More Users → Market Dominance
     ↑                                      │
     └──────────────────────────────────────┘
```

## Platform Strategy

**Hoffman's Platform Principles**:
1. **Liquidity**: Make it easy for supply and demand to meet
2. **Trust**: Identity verification, reputation systems
3. **Network density**: Maximize connections per user
4. **Multihoming costs**: Make it costly to leave platform

**LinkedIn Network Effects Logic**:
- Professional identity is unique and non-transferable
- Your network represents your career capital
- Employers need professionals, professionals need employers
- Network keeps growing without heavy marketing
