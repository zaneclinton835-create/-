# Peter Thiel 著作研究

## 核心著作

### 《从0到1》(Zero to One, 2014)

**与Blake Masters合著，基于斯坦福课程CS183**

**核心论点：**

1. **竞争是失败者游戏 (Competition is for Losers)**
   - "Competition is for losers. If you want to create and capture lasting value, look to the corner of competition."
   - 哈佛/硅谷主流思维认为竞争是健康的——这是20世纪最大的迷思
   - 竞争激烈的市场里，利润会被侵蚀到零。即使赢了，也是Pyrrhic victory

2. **从0到1 vs 从1到N**
   - 从1到N：全球化，复制已知模式，做竞争对手已经做的事
   - 从0到1：科技创新，缔造全新市场，全新的商业模式
   - "每当我面试一位CEO，我问的问题永远是：你做的东西有多少是0到1，有多少是1到N？"

3. **垄断不是坏词**
   - 企业利润来自垄断，垄断者会说竞争存在来保护自己
   - 非垄断者说市场很大会让人怀疑
   - 真正的垄断企业会刻意缩小市场定义来逃避反垄断审查
   - "垄断者与非垄断者的区别：垄断者把产品说成是很多小市场的交叉，非垄断者把产品说成是一个大市场的细分"

4. **垂直进步 vs 水平进步**
   - 水平进步：从1到N，全球化，复制
   - 垂直进步：从0到1，科技，原创
   - 全球化 + 中国崛起 = 从1到N的尽头

5. **秘密 (Secrets)**
   - 世界仍有秘密等待发现
   - 两类秘密：自然的秘密（科学）和人的秘密（社会学）
   - "你还相信有秘密吗？"——这个问题测试你是否还有企业家精神

6. **幂律分布 (Power Law)**
   - 风投的回报不是正态分布，是幂律分布
   - 10%的公司带来90%的回报
   - 分散投资是" institutionalized mediocrity"
   - Founders Fund的"sexy space"投资：SpaceX比100个普通太阳能公司更有价值

### 《竞争不是斗争》(Competition is a Trap, 2017)

**HBR文章，系统阐述竞争的本质**

**核心论点：**
- 竞争是一种意识形态（competition as ideology）
- 竞争把人变成"呆伯特"（Dilbert）：困在零和游戏里
- 竞争使企业关注对手而非用户
- "我们有竞争焦虑"——竞争让我们觉得必须不断竞争
- 竞争优势的4种类型（见下）

### 关键术语（自创）

| 术语 | 含义 |
|------|------|
| Zero to One | 从无到有，原创性创新 |
| Competition is for losers | 竞争是失败者游戏 |
| Monopoly capitalism | 垄断是资本主义的目标 |
| Secrets | 尚未被发现的真理 |
| Creative monopoly | 创造性垄断 |
| The Network Effect | 网络效应（护城河之一） |
| Complexity curse | 复杂性问题（信息不对称的结果）|
| "We wanted flying cars, we got 140 characters" | 对科技行业创新的批评 |
